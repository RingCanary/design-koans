# Visible Signifiers

- Status: draft
- Domain: Shared
- Claim: A-02
- Family: affordance
- Lens: signifiers
- Question: How do users know an element can be used?
- Rule: Every actionable element should expose a visible signifier that signals use.
- Why it works: Discoverability fails before comprehension; users must first see that an action exists.
- Use when: Interactive elements could be mistaken for static content.
- Avoid when: Decorative treatment hides actionability or competes with content.
- Small example: Links stay visibly distinct, primary buttons use clear fill/outline contrast, and icon-only actions add labels or tooltips.
