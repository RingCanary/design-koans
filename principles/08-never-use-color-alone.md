# Never Use Color Alone

- Status: draft
- Domain: Shared
- Claim: A-15
- Family: composition
- Lens: color
- Question: How should state meaning remain legible beyond color?
- Rule: Pair color with text, icon, shape, or position whenever color communicates meaning.
- Why it works: Redundant channels improve accessibility and reduce missed states.
- Use when: Errors, success, warnings, focus, required states, and chart categories.
- Avoid when: A state can only be inferred by hue or saturation differences.
- Small example: Error state uses red plus iconography and message text, not just a red border.
